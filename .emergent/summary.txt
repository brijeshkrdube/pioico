<analysis>**original_problem_statement:**
The user wants to develop a secure, admin-controlled ICO platform for PIOGOLD (PIO), a native coin of the PIOGOLD Mainnet. The platform should allow users to purchase PIO using USDT (BEP20) via Web3 wallets and receive the PIO in the same wallet.

**PRODUCT REQUIREMENTS:**
-   **Blockchain:** PIOGOLD Mainnet (for PIO) and Binance Smart Chain (for USDT).
-   **Pricing:** 1 PIO = 1 gram of gold, with the gold price being manually updatable by an admin.
-   **Purchase Flow:** Users connect their wallet, enter a USDT amount, the system calculates the PIO amount (including discounts), the user sends USDT, and the backend verifies the transaction before sending PIO.
-   **Discounts:** A time-and-amount-based discount system offering bonus PIO.
-   **Referral System:** A 3-level referral system (10%, 5%, 3%) with rewards calculated on the USDT paid.
-   **Admin Panel:** A comprehensive dashboard to manage gold price, ICO status, discounts, view sales, manage the admin wallet's private key, view/manage referrals, and view transaction logs. An emergency stop button is also required.
-   **Frontend:** A responsive UI with wallet connection, live price display, a purchase calculator, transaction status screen, and a referral dashboard.
-   **Security:** The admin's private key must be AES-256 encrypted and stored securely.
-   **Later additions:** The user also requested features for mandatory referrals, a dynamic Meet the Team section, dynamic Legal Documents in the footer, and several new informational sections on the homepage.

**User's preferred language**: English
